## 3) What a “good” vs “bad” print looks like

### Good print (path to $150 → SMA200)

Must clear **all** of:

| Checkpoint | Good |
|------------|------|
| Revenue | ≥ ~$1.81B (clear guide + street) |
| Adj. EPS / rule-of-40 quality | ≥ ~$0.35 and margin stable/expanding |
| U.S. commercial | Growth still ~**100%+** YoY or sequential acceleration called out |
| FY26 guide | **Raise** (midpoint above prior ~$7.66B narrative) |
| Bootcamp / AIP deal metrics | Net adds / remaining performance obligation tone constructive |
| Tone | No warning on gov budget / international sovereignty drag |

**Market reaction template if good:** Gap +6% to +15% toward $140–$150; hold gains 2–5 sessions; SMA200 becomes the medium-term magnet.

### Neutral / “priced in” print

| Checkpoint | Neutral |
|------------|---------|
| Revenue | In-line $1.80–$1.81B |
| Guide | Unchanged or micro-raise |
| Commercial | Strong but “as expected” |
| Reaction | −4% to +4% chop — multiple stays the debate |

### Bad print (path to $110)

Any **one** of these can be enough given valuation:

| Checkpoint | Bad |
|------------|-----|
| Revenue | Miss or only scraping low end of guide |
| U.S. commercial | Deceleration called out / << prior 100% pace |
| FY guide | Flat or cut |
| Margins | Compression from delivery mix / FDE cost optics |
| Gov / politics | Contract risk overshadows numbers |
| Reaction | −8% to −15% Day 0; test $120 then $106–$110 |
